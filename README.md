<h1 align="center">Sales Management System</h1>
<p align="center"><em>A web-based system for managing products, transactions, and sales reports</em></p>

---

## English

A comprehensive web-based sales management system to handle products, sales transactions, and generate reports.

### Features
- Product management (add, edit, delete)
- Sales transactions
- Daily/monthly sales reports
- Automated stock management

### Installation
```
git clone https://github.com/RifkyA911/Sales-Management.git
cd Sales-Management
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate --seed
php artisan serve
```

### Tech Stack
- Backend: Laravel
- Database: MySQL
- Frontend: Blade + Bootstrap
